* =FFMPEG=

** Add music to video with =ffmpeg=
- =ffmpeg -i <video.mp4> -i <audio.mp3> -c copy -map 0:v:0 -map 1:a:0 -shortest output.mp4=
- To add audio at a specific offset in the video:
  - =ffmpeg -y -i <video.mp4> -itsoffset <video offset> -i <audio.mp3> -c copy -map 0:v:0 -map 1:a:0 -shortest output.mp4=
    - Example: =ffmpeg -y -i ganesh_puja_2020.mp4 -itsoffset 00:01:30 -i ganesh_puja_2020_audio.mp4  -c copy -map 0:v:0 -map 1:a:0 -shortest output.mp4=

** Add repeating music to a video with =ffmpeg=
- =ffmpeg -i <video.mp4> -stream_loop -1 -i <audio.m4a> -shortest -map 0:v:0 -map 1:a:0 -c:v libx264 -c:a aac output.mp4=

** Remove audio from video with =ffmpeg=
- =ffmpeg -i <video.mp4> -c copy -an <output.mp4>=

** Trim video with =ffmpeg=
- =ffmpeg -ss <start timestamp> -to <end timestamp> -i <input> -c:a aac -c:v libx264 <output>.mp4=
  - Note that the =-ss= and =-to= parameters have to come /before/ the input parameter

** Extract audio from video
=ffmpeg -i <input video> -vn -c:a aac <output>.aac=

** Swap the order of audio streams with =ffmpeg=
Sometimes videos have the a foreign language as the default audio stream, with English as a secondary stream. To swap the audio streams to make English the default:
#+BEGIN_EXAMPLE
  ffmpeg -i <input file> -map 0:v:0 -map 0:a:1 -map 0:a:0 -disposition:a:0 default -disposition:a:1 none -c copy <output file>
#+END_EXAMPLE
The order of the =-map= parameters determines the order of the streams in the output file

** Downscale video
- To downscale a 4K video to 1080p
  #+begin_example
    -vf scale=1920:1080 -crf 20
  #+end_example

** Reduce video frame rate
- To reduce the frame rate to 60 FPS: =ffmpeg -i <input> -vf "fps​=60" -c:v "h264" -c:a "aac" <output>=

** Increase compression ratio
- =ffmpeg= uses the =-crf= flag to indicate the compression ratio that should be used
- Higher =crf= values indicate more compression, at the expense of more artifacts and slower encoding
- =-crf "18"= is considered high quality
- =-crf "23"= is a compromise between size & quality
- =-crf "28"= prioritizes file size over quality

** Specify multiple video filters
- To specify multiple videos filter, separate the filters with commas
- For example, to downscale a video to 1080p and reduce its frame rate to 60 fps: =-vf "scale=1920:1080,fps=60"=

** Concatenate videos
#+begin_example
  ffmpeg -i "<video_1>.mp4" -i "<video_2>.mp4" -i "<video_3>.mp4" -filter_complex "[0:0][0:1][1:0][1:1][2:0][2:1] concat=n=3:v=1:a=1 [v] [a1]" -vsync 2 -map "[v]" -map "[a1]" "<output>.mp4"
#+end_example
- Make sure that all input videos have identical resolutions
- Make sure that all input videos are encoded as H264/AAC before concatenating
- The =vsync= argument is *very important*

*** To concatenate videos with a fade transition
#+begin_src bash
  ffmpeg -i "<video_1>.mp4" -i "<video_2>.mp4" -filter_complex "[0:v]fade=t=out:st=<start time in sec>:d=<duration in sec>[v1]; [1:v]fade=t=in:st=0:d=<duration in sec>[v2]; [v1][0:1][v2][1:1] concat=n=2:v=1:a=1 [vout] [aout]" -vsync 2 -map "[vout]" -map "[aout]" "<output>.mp4"
#+end_src

** Create a video from a still image and an audio file
#+begin_src bash
  ffmpeg -loop 1 -i "image.jpg" -i "music.aac" -shortest -acodec "aac" -vcodec "libx264" "output.mp4"
#+end_src

** Rotate video
#+begin_src bash
  ffmpeg -i <input video> -vf "transpose=1" -c:a aac -c:v libx264 <output video>.mp4
#+end_src

** Get the duration of a video file
#+begin_src bash
  ffprobe -i <video file> -show_entries format=duration -v quiet -of csv​="p=0"
#+end_src

*** To sum the durations of all videos in a directory
#+begin_src bash
  total=0
  for f in *.mp4; do
      duration=$(ffprobe -show_entries format=duration -v quiet -of csv="p=0")
      total=$(echo "$total + $duration" | bc)
  done
  echo "Total video length: $total seconds"
#+end_src

* Install NodeJS
- First set up [[https://github.com/nvm-sh/nvm?tab=readme-ov-file][NVM]]: =curl -o- "https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh" | bash=
- Activate NVM with: =\. "$HOME/.nvm/nvm.sh"=
- Install NodeJS 24 (LTS) with: =nvm install 24=
- Verify NodeJS version with =node -v=
- Verify NPM version with: =npm -v=
- Reboot
- Now all shells should come with NodeJS version 24 by default

* Access files on an Android device
- Connect the Android device via USB
- Configure the the Android device for USB file sharing
- Use =gio mount -l= to verify that the Android device has been mounted
- Go to =/run/user/$UID/gvfs/=
- The Android device will be mounted as something like ='mtp:host=Google_Pixel_9a_4A251JEBF19625'=
- After you're finished, unmount the device with =gio mount -u "mtp://Google_Pixel_9a_4A251JEBF19625"=

* Configure journald limits to save disk space
- =journald= config file: =/etc/systemd/journald.conf.d/cleanup.conf=
- Configure the following limits:
  #+begin_src conf
    [Journal]
    SystemMaxUse=500M
    MaxRetentionSec=1month
  #+end_src
- Restart =journald= with =systemctl restart systemd-journald=
