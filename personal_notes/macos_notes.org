- To clear terminal scrollback in MacOS: =Edit -> Clear Scrollback=

** Set Terminal Title
- To set the terminal title: =echo -n -e "\033]0;YOUR TITLE HERE\007"=
- Turn this into a zsh function:
  #+BEGIN_EXAMPLE
    function title {
      echo -n -e "\033]0;$@\007"
    }
  #+END_EXAMPLE

** JDK Setup Notes
- To install a JDK from homebrew:
  - Enable the AdoptOpenJDK brew tap: =brew tap AdoptOpenJDK/openjdk=
  - Install OpenJDK 8 with =brew cask install adoptopenjdk8=
- To easily switch JDK versions, add the following to your =.bashrc=
  #+BEGIN_EXAMPLE
    jdk() {
        version=$1
        export JAVA_HOME=$(/usr/libexec/java_home -v"$version");
        java -version
    }
  #+END_EXAMPLE
- AdoptOpenJDK installs its JDKs to =/Library/Java/JavaVirtualMachines/adoptopenjdk-<version>.<jdk|jre>=
  - On my work machine: * =/Library/Java/JavaVirtualMachines/adoptopenjdk-11.jdk= * =/Library/Java/JavaVirtualMachines/adoptopenjdk-8.jdk=
- To configure VS Code for Java development
  - Install the [[https://marketplace.visualstudio.com/items?itemName=vscjava.vscode-java-pack][Java Development Extension Pack]]
  - Configure the JDK 8 runtime
    - Open the settings for =Language Support For Java(TM) by Red Hat=
    - Edit the =runtimes= variable
    - Add the following to the =java.configuration.runtimes= array:
      #+BEGIN_EXAMPLE
        [
            {
                "name": "JavaSE-1.8",
                "path": "/Library/Java/JavaVirtualMachines/adoptopenjdk-8.jdk"
            },
            {
                "name": "JavaSE-11",
                "path": "/Library/Java/JavaVirtualMachines/adoptopenjdk-11.jdk"
            }
        ]
      #+END_EXAMPLE

** Python Setup
- =brew install pyenv= to install the PyEnv Python environment manager
- =pyenv install <latest Python version>=
- Set the global version of Python: =pyenv global <latest Python version>=
- Add the following to your =.zshrc=:
  #+BEGIN_EXAMPLE
    if command -v pyenv 1>/dev/null 2>&1; then
        eval "$(pyenv init -)"
    fi
  #+END_EXAMPLE
- Restart any open shells

** Add italics support to terminal
- Check if your terminal supports italics by running =echo "$(tput sitm)italics$(tput ritm)"=
- If the above text does not show up in italics, add the italics definition to =terminfo=:
  - Create a plain text file called =xterm-256color-italic.terminfo.txt= with the following contents:
    #+BEGIN_EXAMPLE
      xterm-256color-italic|xterm with 256 colors and italic suppport
        sitm~\E[3m, ritm~\E[23m,
        use=xterm-256color,
    #+END_EXAMPLE
  - Run =tic -o ~/.terminfo xterm-256color-italic.terminfo.txt=
  - Set the =TERM= environment variable to =xterm-256color-italic=
