* Mom's Computer Replacement Plan

** High-level process
- Back up Firefox settings
- Back up files to thumbdrive
- Take a screenshot of the desktop
- Get Mom's Microsoft account details
- Remove SSD from old desktop and put it into the external enclosure
- Hook up the new computer
- Install Windows on the new computer
- Create a Microsoft account for Mom (if necessary)
- Install drivers (if necessary)
- Install Office
- Restore Firefox settings
- Restore files
- Restore desktop

** Firefox backup and restore

*** Backup
- Find the Firefox profile directory by going to =about:profiles=
- Back up the profile directory

*** Restore
- Launch Firefox once to create the parent directory structure
- Copy the backed up profile into the profiles directory
- Go into Firefox again, and in =about:profiles=, select "Create Profile"
- Give the profile a name, like "default profile"
- *Select =Choose Folder=*
- Choose the profile that you restored
- Set that profile as the default
