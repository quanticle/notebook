** Create Ubuntu Install Media
- Insert a thumbdrive
- Use =lsblk= to determine what block device corresponds to the thumbdrive
- =umount= any mounted partitions from the thumbdrive (if applicable)
- Run =sudo dd if=<path to Ubuntu image>.iso of=<block device> bs=4M status=progress oflag=sync= to make a bootable thumb drive
