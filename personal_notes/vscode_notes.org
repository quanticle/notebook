** Disable IntelliSense suggestions
- Set ="editor.quickSuggestions"= to =false=

** Kotlin Support
- Install the Kotlin extension by fwcd
- Go into the extension settings and disable the language server, because the language server just adds a bunch of unnecessary configuration complexity

** Use VSCode as git editor
- MacOS
  #+BEGIN_EXAMPLE
    [core]
        editor = "code -w"
  #+END_EXAMPLE
  - To use VSCode Insider
    #+BEGIN_EXAMPLE
      [core]
          editor = "code-insiders -w"
    #+END_EXAMPLE

** Better Markdown Support
- Install =Markdown All In One=

** Vim keybindings
- Install the Vim plugin
- Set =vim.useSystemClipboard= to =true= to make copy and paste more ergonomic
- Also note that the vim plugin has a limit of 1000 results with =/= and =*=
  - /This shouldn't be a problem if I'm just using VSCode for my notes, though/

** Emacs keybindings
- Install the =Emacs= plugin
- =M-x= brings up the command palette
- =C-x C-f= brings up the project file switcher
- =C-x b= brings up the buffer switcher
