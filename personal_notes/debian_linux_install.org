** Create Debian Install Media
- Download the Debian netinstall image from =https://www.debian.org/=
- Insert a thumbdrive
- Use =sudo lsblk= to see which device (=/dev/sd*=) the thumbdrive is
- Unmount the thumbdrive (if applicable)
- =sudo cp <debian_image>.iso <thumbdrive_device>= (note: it's important to copy the image to the device itself (=/dev/sda=, =/dev/sdb=, =/dev/sdc=, etc) instead of a partition on the device (=/dev/sdc1=))
- =sudo sync=

** Get Emacs and Org-Mode

*** Emacs
- Remove pre-installed emacs (if applicable)
  - =emacs-gtk=
  - =emacs-common=
  - =emacs-bin-common=
  - =emacs-common-non-dfsg=
  - =emacs-el=
- Install prerequisites (libraries and development headers for those libraries)
  - =libgtk-3-dev=
  - =xaw3dg-dev=
  - =libxpm-dev=
  - =libpng-dev=
  - =zlib1g-dev=
  - =libjpeg-dev=
  - =libtiff-dev=
  - =libgif-dev=
  - =librsvg2-dev=
  - =libgccjit-12-dev=
  - =libgnutls28-dev=
  - =libncurses-dev=
  - =libjansson-dev=
  - =texinfo=
- Download the latest emacs source from http://ftpmirror.gnu.org/emacs/
- Expand the archive with =tar -xvf "emacs-<version>.tar.xz"=
- =cd "emacs-<version>"=
- Run =./configure --with-native-compilation --with-json --with-pgtk=
- Run =make=
- Run =src/emacs -Q= to verify that the build process produced a runnable emacs
- Run =sudo make install= to install emacs into the system and make it available as a normal application

*** Org-Mode
- Get org-mode sources from: =git://git.sv.gnu.org/emacs/org-mode.git=
- Clone into =~/emacs-org/=
- Run make
