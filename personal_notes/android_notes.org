#+TITLE: Android Notes

* Useful tips

** Prevent Google Fi VPN from disconnecting
- Disable battery optimization for the "Google Communication Services" app

** Save battery by preferring LTE to 5G
- Settings → Network & Internet → Internet → Google Fi → Preferred Network Type

** Save battery by disabling background scanning for wi-fi and Bluetooth
- Location → Location Services
  - Google Location Accuracy → Off
  - Wi-Fi scanning → Off
  - Bluetooth scanning → Off
