* Clean Install

** Make Backups
- Back up Downloads directory

** Create Fedora Install Media
- Install the Fedora Media Writer: =flatpak install flathub org.fedoraproject.MediaWriter=
- Run Fedora Media Writer from the application menu

** Install Fedora

** Install Cascadia Code fonts
- Download Cascadia Code from =https://github.com/microsoft/cascadia-code/releases=
- Install the individual =.otf= files

** Restore =.bashrc=

** Restore SSH and GPG keys
- Copy SSH key from =~/thumbdrive/laptop_ssh_keys/<latest_backup>/ssh/= to =~/.ssh/=

- Set file permissions to 600
- Copy GPG key from =~/thumbdrive/gpg_keys/<latest_backup>/gnupg/= to =~/.gnupg/=
- Set directory permissions to 700
- Set file permissions to 600

* Upgrades

:HTML_CONTAINER_CLASS: page-break-before

** Routine Upgrades
- =sudo dnf upgrade --refresh=
- Unmount external drives
- Reboot

** Upgrading to a new release
- =sudo dnf upgrade --refresh=
- If not installed, install the =dnf= system upgrade plugin with: =sudo dnf install dnf-plugin-system-upgrade=
- =sudo dnf system-upgrade download --releasever​=<release #>=
  - May have to pass the =--allowerasing= flag if some packages have unsatisfied dependencies
- Switch to a console =Ctrl-Alt-F3=
- Run the upgrade with =sudo dnf system-upgrade reboot=
- After the upgrade completes, remove retired packages
  - =sudo dnf install remove-retired-packages= (only needs to be done once)
  - =sudo remove-retired-packages=
