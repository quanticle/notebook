** Autosave
- Set =save_on_focus_lost= to =true=

** Jump To Matching Bracket
- =Ctrl-M= (note that this is the same on both MacOS and Windows)

** Set a ruler for a particular file type
- =Preferences= - =Settings - More= - =Syntax Specific - User=
- Add a ="rulers": [val1, val2, ...]= for all the rulers you want to define

** Disable git gutter
- Set ="show_line_annotation"= to =false=
- Set =mini_diff= to =false=

** Better Markdown Support
- Install the =MarkdownEditing= package
- =Alt-X= (=Option-X=) to toggle checkboxes
- To disable link URL folding set =mde.auto_fold_link.enabled= to =false=
- To keep Markdown editing from messing with your list bullets, set =mde.list_indent_bullets= to =["-"]=

** Ensure Files Always End With a Newline
- Set =ensure_newline_at_eof_on_save= to =true=
- This is necessary for =MarkdownEditing= to continue lists correctly when they're at the end of the file
